## 🔧 如果推送失败

### 方案 A: 排除 python313

如果仓库太大无法推送，可以排除 python313：

```bash
# 更新 .gitignore
echo "python313/" >> .gitignore

# 重新提交
git rm -r --cached python313
git add .gitignore
git commit -m "排除 python313 文件夹"
git push
```

然后在 README 中说明用户需要自己配置 Python 环境。

### 方案 B: 使用 Git LFS

```bash
# 安装 Git LFS
git lfs install

# 追踪大文件
git lfs track "python313/**"

# 提交并推送
git add .gitattributes
git commit -m "使用 Git LFS 管理 python313"
git push
```
